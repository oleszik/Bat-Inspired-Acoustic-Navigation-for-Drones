# Datasets

This folder stores generated synthetic datasets used by acoustic mapping and navigation experiments.

Typical contents include:
- synthetic echo waveforms
- spectrograms
- regression/classification labels
- derived features (e.g., correlation vectors, peak features)

These files are generated by scripts in:
- `signal_processing/`
- `simulation/phase2_mapping/echo_dataset_generator.py`

Because dataset files can be large, they are treated as generated artifacts.
